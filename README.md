# Oil Impact on Foreign Exchange Markets: Automated PPT Generator

## 🌟 Revolutionizing Financial Presentations with Code

This project pioneers an innovative approach to financial education by **automating PowerPoint presentation creation** using JavaScript. Instead of manually crafting slides, we generate dynamic, visually stunning presentations that explore the intricate relationship between crude oil prices and India's foreign exchange market.

### 🚀 What's New & Exciting

- **Code-Driven Presentations**: Write JavaScript to build entire PPT decks programmatically
- **Financial Data Visualization**: Automated charts and analysis slides for economic insights
- **Custom Design System**: Professional color schemes and layouts tailored for business education
- **Instant Generation**: Run a script to create polished presentations in seconds
- **Educational Innovation**: Bridging programming and finance for modern business students

### 📊 Project Overview

This repository contains JavaScript code that generates comprehensive PowerPoint presentations analyzing how fluctuations in crude oil prices affect India's foreign exchange markets. The presentations include:

- Market analysis slides
- Economic impact visualizations
- Data-driven insights
- Professional business layouts

### 🛠️ Tech Stack

- **pptxgenjs**: Powerful JavaScript library for PPT generation
- **Node.js**: Runtime environment for executing the scripts
- **Custom Design System**: Navy, gold, and teal color palette for professional appearance

### 🎯 Key Features

- **Automated Slide Creation**: Functions for titles, content, charts, and dividers
- **Consistent Branding**: Custom color scheme and typography
- **Responsive Layouts**: 16:9 aspect ratio optimized for modern displays
- **Footer Integration**: Automatic addition of academic branding
- **Shadow Effects**: Professional visual enhancements

### 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/amanjot-a/Oil_impact_FX.git
   cd Oil_impact_FX
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the PPT generator**
   ```bash
   node ppt--3.js
   ```

4. **Find your generated presentation** in the project directory!

### 📁 Files Included

- `ppt--3.js` - Main presentation generator script
- `PPT.js` - Alternative presentation configurations
- `Written.js` - Additional content scripts
- Generated PPT files showcasing different analysis approaches
- Package configuration for easy setup
